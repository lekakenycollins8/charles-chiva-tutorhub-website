# Chiva TutorHub Website Project Overview

## Project Description
Chiva TutorHub is a professional tutoring services website designed to showcase tutoring services in Chemistry, Mathematics, Business Studies, and Accounting. The website aims to provide a modern, user-friendly interface for students seeking tutoring services, with features for resource sharing, blog content, and direct communication with the tutor.

## Key Features
- Professional presentation of tutoring services
- Resource library with both free and paid materials
- Blog section for educational content
- Contact system with chat functionality
- Payment integration for services and premium resources
- Admin dashboard for content management

## Technology Stack
- **Frontend**: Next.js 15, React 19, Tailwind CSS, shadcn UI
- **Backend**: Next.js API routes
- **Database**: MongoDB Atlas with Prisma ORM
- **State Management**: Zustand (if needed)
- **Authentication**: Admin-only authentication
- **File Storage**: Hybrid approach with MongoDB GridFS and Cloudinary
- **Payment Processing**: Stripe

## Project Goals
1. Create a professional and visually appealing website that converts visitors to clients
2. Provide easy access to information about tutoring services
3. Enable resource sharing with both free and paid options
4. Facilitate direct communication between potential clients and the tutor
5. Allow for easy content management by the site administrator

## Target Audience
- High school and college students seeking tutoring in Chemistry, Mathematics, Business, and Accounting
- Parents looking for tutoring services for their children
- Educational institutions seeking partnership opportunities
